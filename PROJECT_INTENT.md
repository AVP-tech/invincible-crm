# Project Intent

Build a startup-grade MVP for an AI-first "Invisible CRM" that helps very small businesses manage relationships and follow-ups with almost no manual data entry.

Primary promise:
- The CRM should work for the user instead of making the user work for the CRM.

Core interaction:
- Users type a natural-language update.
- The system extracts structured CRM actions.
- The user confirms or edits the preview.
- The CRM stores contacts, deals, tasks, notes, and activities automatically.
